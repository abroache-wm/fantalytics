Power Rankings

Algorithm-based rankings considering multiple factors

Power Ranking Algorithm

The Power Score is calculated using the following weighted factors:

  • Championships (25%) - Each championship adds significant value
  • Win Percentage (20%) - Overall winning percentage across all seasons
  • Points Per Game (20%) - Average points scored per game
  • Playoff Success (15%) - Playoff appearances and performance
  • Consistency (10%) - How consistent scoring is week to week
  • Recent Form (10%) - Performance over the last 3 seasons

Formula: Power Score = (Championships × 25) + (Win% × 20) + (PPG/120 × 20) + (Playoff Rate × 15) + (Consistency × 10) + (Recent Win% × 10)

Luck Analysis

How lucky or unlucky has each owner been?

Luck Index Algorithm

The Luck Index measures the difference between actual wins and expected wins:

  • Expected Wins - For each game, we calculate what percentage of all teams that week you would have beaten with your score
  • Actual Wins - Your actual win/loss record
  • Luck Differential - Actual Wins minus Expected Wins
  • Positive values indicate you've been lucky (won more than expected)
  • Negative values indicate you've been unlucky (won less than expected)

Example: If you scored 120 points in Week 1 and that would have beaten 8 out of 12 teams, your expected wins for that week is 0.67
